## Selected Work

### AquaSens — Smart Irrigation Prediction System
> TypeScript · React · Node.js · Flask · MongoDB · scikit-learn

An intelligent, full-stack irrigation decision support system. A Decision Tree model trained to 99.7% accuracy predicts irrigation requirements — Low, Medium, or High — from environmental and soil parameters, with full decision path transparency and agronomist-grade recommendations. Deployed across three independent services: a React/TypeScript frontend, an Express REST API, and a Flask ML microservice.

[View repository](https://github.com/HoussemEddineChaouch/AquaSens)

---

### ChefsEye — Food Recognition via Computer Vision
> JavaScript · Transfer Learning · Computer Vision

Users upload a food image; the system classifies the dish and returns similar meal recommendations. Built on transfer learning to enable real-time prediction from user-submitted photos.

[View repository](https://github.com/HoussemEddineChaouch/ChefsEye)

---

### OptiRoute — Interactive TSP Solver
> JavaScript · React · Graph Algorithms

A browser-based graph editor that lets users build weighted graphs and solve the Travelling Salesman Problem in real time using a brute-force algorithm, with a live visual of the optimal route.

[View repository](https://github.com/HoussemEddineChaouch/OptiRoute)

---

### Fluxo — Team Task Management Platform
> PHP · MySQL · Bootstrap 5 · Chart.js

Role-based task management for Admins and Team Members. Includes assignment workflows, progress dashboards powered by Chart.js analytics, and clean separation between management and contributor views.

[View repository](https://github.com/HoussemEddineChaouch/Fluxo)

---

### QuranAudio — Quran Streaming Player
> JavaScript · REST API

A clean, browser-based audio player for streaming Quranic recitations via public API. Supports verse browsing, chapter navigation, and continuous playback.
